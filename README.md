  <h1>OSforGH-Lab</h1>

  <p>My learning repository for OSforGH, sharing Grasshopper workflows and example files for structural analysis experiments. I will be uploading here all the Grasshopper files I work on and analyze using OSforGH.   <h2>About OSforGH</h2>
  <p>
    <strong>OSforGH</strong> is a Grasshopper plugin built on <strong>OpenSees</strong>, providing tools for structural and earthquake engineering analysis. 
    This repository contains my personal learning files and workflows using OSforGH.
  </p>

    <li><strong>Official Repository:</strong> <a href="https://github.com/strdesigner/OSforGH" target="_blank">https://github.com/strdesigner/OSforGH</a></li>
